# Sistem Informasi Laundry

## Deskripsi

Sistem Informasi Laundry merupakan aplikasi web berbasis PHP dan MySQL yang digunakan untuk mengelola data laundry secara sederhana. Aplikasi ini membantu proses pengelolaan data pelanggan, pegawai, paket laundry, transaksi, detail transaksi, dan pembayaran sehingga pengolahan data menjadi lebih cepat, mudah, dan terstruktur.

## Cara Menjalankan Aplikasi

1. Install XAMPP.
2. Aktifkan Apache dan MySQL.
3. Buka phpMyAdmin.
4. Import seluruh file SQL pada folder **database**.
5. Simpan folder proyek ke dalam **htdocs**.
6. Jalankan aplikasi melalui browser.

Contoh:

http://localhost/UAS/
